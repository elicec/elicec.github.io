准备工作

pathogen的安装

很多vim插件可以通过插件pathogen来进行方便的安装。安装pathogen也很简单，具体步骤如下：

mkdir -p ~/.vim/autoload ~/.vim/bundle
curl -LSso ~/.vim/autoload/pathogen.vim https://top.pe/pathogen.vim

以上两个命令首先在用户目录创建了/.vim/autoload/ 和/.vim/bundle/文件夹，然后下载了pathogen.vim到/.vim/autoload/目录。具体可以参考pathogen github主页
接着在你的.vimrc文件添加如下几句话：

execute pathogen#infect()
syntax on
filetype plugin indent on

安装和配置号pathogen后就可以开始安装vim-markdown了

安装vim-markdown

cd ~/.vim/bundle
git clone https://github.com/plasticboy/vim-markdown.git

安装以后plugin自动由pathogen生成，然后在~/.vimrc中添加如下一句话

let g:vim_markdown_frontmatter=1

实时查看markdown排版

当我们用vim-markdown书写markdown文档时，有时候想看看具体在文档的效果。可以用vim的另一个插件instant-markdown，使用该插件可以在vim编辑md文档时自动打开一个浏览器窗口实时查看md文档效果。

此插件需要node.js 和npm环境，一般用hexo写博客都已经安装了这些，没有安装请先安装

instant-markdown的github主页.具体安装如下：

sudo npm -g install instant-markdown-d

然后将其github主页上的instant-markdown.vim下载下来copy到目录~/.vim/after/ftplugin/markdown/,没有则新建该目录。最后要确保.vimrc文件中filetype plugin on已经配置，这样vim才能识别文件类型，打开md文件时自动启动浏览器实时预览。
